# Amazon Clone App

A simplified clone of the Amazon website developed using HTML and CSS.



## Features

- [ ] Homepage with product listings.
- [ ] Product detail pages.
- [ ] Shopping cart functionality.
- [ ] Simulated checkout and order process.

## Technologies Used

- HTML5
- CSS3
